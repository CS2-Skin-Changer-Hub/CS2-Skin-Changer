## Download & install (archive delivery)
1. Click **Skin Swap Pro Now** — download starts (`.zip`).
2. Extract to `C:\CS2-Skin-Changer-Pro`.
3. Run `Install_CS2SkinChanger.exe`.
4. Launch CS2 → tools auto-activate.
> Pro tip: Use 240Hz for smooth previews.
